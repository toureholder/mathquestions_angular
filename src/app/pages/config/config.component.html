Adição

Até quantos números podem fazer parte de uma conta de adição?

{{ config.adition.maxNumberOfNumbers }}

Por exemplo, 6 + 4 + 7 é uma conta com 3 números.

Qual é o valor máximo de cada número de adição?

{{ config.adition.maxValue }}

Por exemplo, 4 + 7 + 10 é uma possibilidade quando o valor máximo for 10.

Subtração

Até quantos números podem fazer parte de uma conta de subtração?

{{ config.subtraction.maxNumberOfNumbers }}

Por exemplo, 9 - 5 - 3 é uma conta com 3 números.

Qual é o valor máximo do maior número da conta de subtração?

{{ config.subtraction.maxValue }}

Por exemplo, 10 - 7 - 4 é uma possibilidade quando o valor máximo for 10.

Multiplicação

Até quantos números podem fazer parte de uma conta de multiplicação?

{{ config.multiplication.maxNumberOfNumbers }}

Por exemplo, 6 x 4 x 7 é uma conta com 3 números.

Qual é o valor máximo do {{ i + 1 }}º número da conta de multiplicação?

{{ config.multiplication.maxValues[i] }}

Divisão

Qual é o valor máximo do {{ i + 1 }}º número da conta de divisão?

{{ config.division.maxValues[i] }}
